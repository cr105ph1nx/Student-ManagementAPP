# JAVA Student Management App

## Setup Instructions
### 1. Change the extension of setup.jpg to setup.jar
### 2. Run script.sql in your oracle environement like so:       
`SQL> @C:/(path)/script.sql`                                 
(Where 'path' is the path to the file in your machine)
### 3. Execute setup.jar
